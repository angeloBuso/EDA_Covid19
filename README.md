# EDA_Covid19
 Panorama dos dados da Covid19 no Brasil e no Mundo.

Dados analisados de 01/01/2020 a 18/04/2021.

Os dados do dataSet utilizado são medidos diariamente, o que enseja em sua desatualização rápida. No entanto se rodar o notebook obterá informações atualizadas.
